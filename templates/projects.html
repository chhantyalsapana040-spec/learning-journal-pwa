Temperature Converter PWA

The Temperature Converter PWA is a simple and responsive Progressive Web App that allows users to convert temperatures between Celsius, Fahrenheit, and Kelvin. It is designed with a mobile-first approach and can be installed on any device for offline use.

This project helped me understand the fundamentals of frontend development, including layout structure, CSS styling, and preparing a web app for PWA capabilities.
